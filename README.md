# Printing Press
Version 1.01

Adds a printing press node which copies books at the cost of three paper

Crafting:
```
W = wood
M = metal ingot

press:
W,W,W
W,M,W
W,M,W
```

## Changelog:

1.0: initial version with printing press
1.01: copies have different names from original
